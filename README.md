# Jake-And-The-Secret-Key
A colorful 2D platformer developed in Python using the Pygame library. This project features custom sprite animations, a life/health system, and a progression mechanic where the player must find a hidden key to trigger a final Boss Fight.

Key Features:

Action Gameplay: Shoot enemies and avoid obstacles.

Economy System: Collect coins scattered throughout the candy-themed levels.

Boss Battle: A unique feline encounter waiting at the end of the journey.

Retro Vibe: Inspired by classic platformers with a modern "Adventure Time" aesthetic.
